Preferences

You can change the configuration of the editor here, although after applying your changes, you must refresh all instances of the editor for the changes to take effect.

Autocompletion

jsCompletionScope:

The scope potential JavaScript completions are collected from. Only has an effect when jsCompletionSource is set to "scope".

jsCompletionSource: ""

The source to get JavaScript completions from. Can be set to "keywords" or "scope".

Indentation

indentUnit: ""

The unit in which indentation happens. Must consist of either the same amount of spaces as tabSize, or a single tab character.

tabSize:

The size of a tab, in columns.

Key binds

indentWithTab:

Allows the user to press Tab to insert the indentUnit characters at the cursor, or if something is selected, indent the selection. Shift-Tab removes a unit of indentation from all selected lines.

vscodeKeymap:

Whether the editor uses the default CodeMirror 6 key binds or the Visual Studio Code key binds.

deprecated Use the other key bind options instead.

Extensions

abbreviationTracker:

Allows you to type CSS-like Emmet abbreviations in the code, and expand them into HTML and CSS code, by pressing Tab.

colorPicker:

Shows a color picker next to CSS color values.

imagePlaceholders:

Allows the user to drag images in the editor from a file viewer, and see them as actual images in the code.

indentationMarkers:

Shows indentation markers for users to easily see how much lines are indented.

interact:

Allows the user to interact with certain JavaScript values, by pressing "Alt" and clicking and dragging with the mouse.

Warning: When enabling this option on mobile devices, the editor may not behave as expected.

rectangularSelection:

When holding down the Alt key and left mouse dragging, rectangular selections will be created.

Note: When this extension is enabled, multi-selections will be made by holding down the Ctrl key, NOT the Alt key.
