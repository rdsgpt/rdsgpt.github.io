About the Site

This is the site for the latest version of the DrRcraft Code Editor. It has many tools that will (can) upgrade your code. Currently, the latest version of the editor is version 1.6.

Browser Support

This Code Editor has two versions, v1 and v2. Version 1, what you are on right now, is fully supported in the latest versions of all browsers:

Version
v1.6 Yes Yes Yes Yes Yes Yes

Meanwhile, version 2 is supported in the latest versions of all major browsers:

Version
v2.3 Yes No Yes Yes Yes Yes

Version 2 is not supported in Internet Explorer 11 and earlier versions.

If you use IE11 as your main browser and require support for it, use version 1. It is the most stable release, and is still supported for bugfixes and accessibility changes. However, no new features will be added to the editor.
