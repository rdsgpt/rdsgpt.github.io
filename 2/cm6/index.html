Keyboard Shortcuts

Because the default context menu is replaced with a custom one, you need to use some keyboard shortcuts to perform basic functions. Many, if not all of these shortcuts, are listed below.

Description Windows macOS
Select all text Ctrl + A Cmd + A
Copy text Ctrl + C Cmd + C
Adjust text indent Ctrl + G Cmd + G
Paste text Ctrl + V Cmd + V
Cut text Ctrl + X Cmd + X
Redo action Ctrl + Y or Shift + Ctrl + Z Shift + Cmd + Z
Undo action Ctrl + Z Cmd + Z
Run code Ctrl + Enter Cmd + Enter
Indent less Shift + Tab
Indent more Tab
