About the Site

This is the site for the latest version of the DrRcraft Code Editor. It has many tools that will (can) upgrade your code. Currently, the latest version of the editor is version 2.3.5, which updates a component.


Changelog for v2.3.6 (2022 February 20)

  • Updated CodeMirror from 5.65.11 to 5.65.12

Browser Support

This Code Editor has two versions, v1 and v2. Version 1 is fully supported in the latest versions of all browsers:

Version
v1.6 Yes Yes Yes Yes* Yes Yes

Meanwhile, version 2 is supported in the latest versions of all major browsers:

Version
v2.3 Yes Yes Yes No Yes Yes

Version 2 is not supported in Internet Explorer 11 and earlier versions.


Internet Explorer 11

Version 2 cannot be opened by Internet Explorer 11. If you ever need support for Internet Explorer 11, you may use version 1, the previous major release. It is still supported for minor bug fixes, but no new features will be added.

Note: Internet Explorer does not support version 1's Export feature.
