DrRcraft Code Editor

The DrRcraft Code Editor is a professional online IDE that has the tools you need. With this editor, you can make a website, a game, a useful tool, a messaging app, anything you can dream of.

Features

  • Mobile Support

    Works well for phone and tablet users.

  • Line Numbers

    Shows gutters on the left side of the editor, with line numbers and other information.

  • Autocompletion

    Gives the user completion hints to avoid typing too much.

  • Search/Replace

    Editor and RegExp search, and replace functions.

  • Performance

    Responsive even with huge documents.

  • Syntax Highlighting

    Clear highlighting for users to understand the code better.

  • Code Folding

    Collapse content to make the code easier to read.

  • Bracket and Tag Closing

    Automatically insert matching tags or brackets when typing.

  • CSS Color Picker

    Pick a color, and automatically set its exact hexadecimal code.

  • Emmet Abbreviations

    Write CSS-like Emmet abbreviations in the HTML, and expand it into markup.

  • Image Placeholders

    Drag images from a file viewer inline, and view them easily.

  • Indentation Markers

    View exactly how much a line of code is indented.

  • Interactions

    Interact with values and change them by clicking and dragging.

  • Rectangular Selections

    Hold Alt and drag to create a rectangular selection in your code.

  • Sharing

    Get a permanent link to share your code with others.

  • Running in Fullscreen

    Test your apps and games in a fully immersive experience.

  • History

    Revert accidental changes by going back in time.

  • Database

    Store your items in a place which you can access at any time.

  • AWS Deployments

    Create a production build of your project so people can use it without getting interrupted by frequent changes.
